# job_application_builder
Python automation scripts for generating tailored job application documents.

Created a buildprocess for applications and removed repetition.

# How to use.

There a two main files.

- config.yaml
Source of information about you and the application target.
Just follow the example given.


- anschreiben_text.txt
This is just a plain text document which gets embeded into the
application letter by the build pipeline.
